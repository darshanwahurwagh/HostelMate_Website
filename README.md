# HostelMate

HostelMate is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed to simplify and automate hostel management operations for educational institutions. The platform provides separate dashboards for administrators and students, enabling efficient management of rooms, attendance, complaints, fees, and hostel services.

## Features

### 👨‍💼 Admin Module
* Dashboard with hostel statistics
* Room Management (Add, Update, Delete Rooms)
* Student Management
* Room Allocation Approval System
* Attendance Monitoring
* Complaint Management
* Fee & Mess Bill Management
* QR Code Attendance Tracking
* Email Notification System
* View Hostel Analytics

### 🎓 Student Module
* Student Dashboard
* Apply for Room Allocation
* View Allocated Room Details
* Check-In / Check-Out Attendance
* QR Code Attendance Scanner
* Submit Complaints
* View Complaint History
* View Attendance History
* View Fee & Mess Bill Details
* Online Payment Integration Ready
